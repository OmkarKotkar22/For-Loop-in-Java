## ðŸ”„ General Loop Operations

This section focuses on iterative programming, demonstrating how to solve common number and sequence problems using **For**, **While**, or **Do-While** loops.

### Key Concepts

* **Iteration:** Repeating a block of code until a condition is met.
* **Modulus and Division:** Using `%` and `/` for digit-by-digit processing in number algorithms.
* **Flag Variables:** Using boolean or integer variables to mark if a property (like primality) is true.

### 1. Number Properties & Checks

These files implement algorithms to check for specific mathematical properties.

| File Name | Description |
| :--- | :--- |

| `prime_no_for_loop.java` | Prints prime numbers up to a given limit. |
| `check_armstrong_no_for_loop.java` | Checks if a number is an **Armstrong number**. |

| `sum_of_primenoforloop.java` | Calculates the sum of prime numbers (using while/do-while loop). |

## ðŸ’» Getting Started

### Prerequisites

* **Java Development Kit (JDK 8+)**: Ensure you have a recent version of the JDK installed.

### Running the Code

1.  **Clone this repository:**
    ```bash
    git clone [Repository URL]
    ```

2.  **Navigate to the directory:**
    ```bash
    cd [repository-name]
    ```

3.  **Compile any Java source file** (e.g., for `solid_diamond.java`):
    ```bash
    javac solid_diamond.java
    ```

4.  **Execute the compiled class:**
    ```bash
    java solid_diamond
    ```
